## Current Focus

### Vibe Guider — Work in Progress

[Vibe Guider](https://github.com/ShahanMalik/vibe_guider) is an AI-powered software project advisor that converts a project idea or technical question into structured guidance.

It uses a multi-agent workflow to:

- Understand project requirements
- Ask smart clarification questions
- Recommend technologies and packages
- Design software architecture
- Identify risks and implementation concerns
- Gather useful technical resources
- Produce a structured final development guide
- Stream workflow progress through a FastAPI endpoint
- Generate downloadable project-structure ZIP bundles

> **Important:** Vibe Guider is still under active development. It is an experimental prototype and is not yet production-ready. Authentication, tests, security hardening, deployment configuration, monitoring, persistent history, and other important features are still incomplete.

**Main technologies:** Python, FastAPI, OpenAI API, LangGraph, LangChain Core, Pydantic, FastMCP
